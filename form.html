How it works

  1. I build a free initial mockup so you can see what your site will look like before committing to anything.
  2. If you're happy to proceed, you pay a 20% deposit to begin the build.
  3. We iterate together until you're fully happy with the result.
  4. Once you accept the final version, you pay the remaining balance.
  5. I ship the site live and hand over all code and hosting credentials, so you fully own and can manage everything going forward.
  6. Any changes requested after acceptance are billed at $100/hour, with a 1 hour minimum.

How it's built & hosted

Your website is written in standard HTML and the code is stored in an online code repository, so it can be read, exported, or modified by any developer in future — you're never locked in to me. It's hosted on Vercel, a reliable hosting platform with no cost for basic sites like yours.
